# Code of Conduct

The GovForge project adopts version 2.1 of the **Contributor Covenant**
as its Code of Conduct, without modification.

The full text — including community standards, enforcement guidelines,
and reporting process — is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Our pledge

We pledge to make participation in this project a respectful experience
for everyone, regardless of background or identity.

## Scope

This Code of Conduct applies to all project spaces — the GitHub repo
(issues, pull requests, Discussions, code review), the marketing site,
any community channels (Discord / Slack / forums) operated by the
project, and any in-person event where someone is officially
representing GovForge.

## Reporting

Concerns about conduct should be sent privately to:

- **Email**: [conduct@govforge.dev](mailto:conduct@govforge.dev)

All reports are reviewed by the project maintainers and treated as
confidential. We aim to acknowledge within 72 hours.

The Contributor Covenant's
[enforcement ladder](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)
is the default escalation framework.
